### 3. Jodie Prompt Improvements

Additive changes only — no structural change to the check-in flow. These improve the quality of data collected in every check-in and directly improve the Weekly Reflection.

| Addition | What it captures |
|---|---|
| "What's one specific thing that happened this week?" | Named, dateable events |
| Sleep/energy probe (if not raised naturally) | Strongest single wellbeing predictor |
| Positive anchor before close | Wins data — currently underrepresented |
| Social connection probe | Strong predictor, currently underrepresented |
| "Thinking back across the whole week..." framing | Weekly perspective rather than just today |

Requires prompt version bump and regression testing. University first (`jodie_prompt_v2_uni.md`), then Pro (`jodie_prompt_v2_pro.md`).
