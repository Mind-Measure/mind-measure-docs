## Tech Stack

- **Frontend**: React + Vite + TypeScript (admin dashboards, mobile apps)
- **Documentation**: Next.js 14 + Nextra (this site)
- **Database**: AWS Aurora Serverless v2 (PostgreSQL)
- **Authentication**: AWS Cognito (students/users), custom JWT (superuser 6-digit code)
- **AI Analysis**: AWS Bedrock (Claude 3.7 Sonnet) — transcript analysis, risk detection
- **Voice Agent**: ElevenLabs Conversational AI — Jodie check-in agent
- **Email**: AWS SES from noreply@mindmeasure.co.uk
- **Deployment**: Vercel (auto-deploy from GitHub on push to `main`)
